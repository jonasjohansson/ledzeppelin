# Importing from LEDger

> **Status: outline — draft in progress.** Bringing a rig definition in from the LEDger
> companion.

## What this page will cover
- What **LEDger** is (the import companion; formerly "Kagora").
- Triggering import from the top-bar **Import from LEDger…** button → it opens the **Inventory
  popout** and a file picker.
- Picking a preset, **assigning/auto-filling controller IPs**, and applying.
- **Important behaviour:** an import **replaces the current rig** (devices/fixtures) — it is
  not additive — and is **not undoable**, so save/snapshot first.
- How the main window updates automatically once you apply (live sync).

_See also: [Fixtures & the Inventory](05-fixtures-and-inventory.md), [Devices](04-devices-and-scanning.md)._
